# Key Mapping Convention

- Space: custom text/code operation
    - rn: rename
    - ca: code action 
    - h: no search highlight
    - dox: generate doxygen header
    - cl: line comment
    - c: comment operator
    - Space: find old files
    - fg: live grep
    - fh: find help tags
    - k: show code API

- Ctrl: operation that change window or tab
    - n: open file tree
    - p: file search

- Alt: switch window/tab
    - w: window
    - t: tab

- g: goto
    - d: definition
    - i: implementation
    - r: reference
    - t: type definition
    - x: diagnostics
    - s: document symbol
    - S: workspace symbol

# Custom Commands

- qt: clost tab
- gdiff: show git diff with file tree
- ghist: show git diff with commit history
